# GptCopilot

## Run tasks

Within the gpt-copilot workspace, you can run tasks for the projects defined in the `apps` and `libs` directories. e.g:

```sh
nx serve apps/app
nx serve apps/api
```

To see all available targets to run for a project, run:

```sh
nx show project gpt-copilot
```
